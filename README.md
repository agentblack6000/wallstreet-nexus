# Exploring Market Regimes with an adaptive portfolio

## Visualizing the dataset

The results of the visualization of the Kenneth R. French 30 Industry Portfolios can be seen in the ```graphs``` directory, and a correlation heatmap

### Usage

```zsh
usage: python visualization.py [-h] [-f FILENAME] [-g] [-c]

Using Pandas, matplotlib, Numpy, and Seaborn to visualize monthly returns and correlations

options:
  -h, --help
  -f, --filename FILENAME
  -g, --graphs
  -c, --correlations
```
